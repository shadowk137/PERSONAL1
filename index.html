☁️ Google Drive Sync (Multi-Device)
Connect this app to a small free Google Apps Script "API" backed by a file on your Google Drive. Once connected, every device using the same link below will read and write the same data — add a tenant on your phone, see it instantly on your laptop.
One-time setup (takes ~5 minutes, see README for full steps):
1. Go to script.google.com → New project
2. Paste in the provided Code.gs script
3. Deploy → Web app → Execute as Me, Access: Anyone
4. Copy the generated …/exec URL and paste it below
Not connected — data is only saved on this device.
⚠️ Anyone who has this Apps Script URL can read and write the data directly (there's no per-user login on the API itself) — only share it with people you trust with edit access.
📦 Storage Overview
Data is cached in this device's localStorage for instant loading and offline use, and — once connected above — synced to a JSON file on Google Drive so every device stays in agreement.
localStorage Keys used:
  • rms-tenants-v3 — Tenant records (local cache)
  • rms-payments-v3 — Payment records (local cache)
  • rms-auth — Login credentials
  • rms-api-url — Your connected Apps Script URL
